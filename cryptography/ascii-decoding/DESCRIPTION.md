For example, to decode the ASCII values `87 79 82 76 68`, we find the numbers in the ASCII table and replace them with the correct character: 
- 87 → **W**  
- 79 → **O**  
- 82 → **R**  
- 76 → **L**  
- 68 → **D**  

So, the decoded message is: `WORLD`
